# Orion Platform

Orion Platform es una plataforma moderna self-hosted diseñada para aprender y aplicar prácticas reales de DevOps y SRE sobre infraestructura Linux on-premise.

El proyecto simula un entorno de producción real utilizandoAPIs modernas y automatización operativa.

## Objetivos del Proyecto

* Construir una plataforma escalable y segura desde cero.
* Aprender DevOps mediante experiencia práctica real.
* Implementar buenas prácticas de infraestructura, CI/CD y observabilidad.
* Documentar incidentes, decisiones arquitectónicas y procesos operativos.
* Simular operaciones reales de producción sobre un servidor Ubuntu.

## Enfoque del Proyecto

Este repositorio no busca únicamente desplegar aplicaciones.

El objetivo principal es construir una plataforma operable, monitoreada, automatizada y resiliente, documentando durante el proceso:

* Incidentes reales
* Postmortems
* Runbooks operativos
* Decisiones arquitectónicas
* Troubleshooting
* Automatización
* Estrategias de recuperación
